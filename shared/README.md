# Shared API Keys

API keys are stored in a single `localStorage` entry (`saasApiKeys`) and shared across all Saas apps:

- **PitchDeck** – OpenAI, Unsplash, Pexels, Pixabay, Google Client ID
- **InfoGraphics** – Giphy, Pixabay, Pexels, OpenAI
- **ColorWriter** – OpenAI
- **StoryWriter** – OpenAI, Unsplash

Enter your keys once in any app’s Settings; they will be available in all apps.

**Security:** Keys are stored locally in your browser only. They are never sent to our servers—only to the APIs you use (OpenAI, Unsplash, etc.).
