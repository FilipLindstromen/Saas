# PostIt – Plan & Brainstorm

A visual planning and brainstorming tool with post-it notes, comments, and hand-drawn style arrows. Great for funnels, offer maps, and idea mapping.

## Features

- **Notes (post-its)** – Add by double-clicking on the canvas. Drag to move, double-click text to edit. Resize from the bottom-right corner. Soft gradient backgrounds and pushpin style like the reference.
- **Snap mode** – Toggle in the toolbar to soft-snap notes and comments to a grid in X and Y.
- **Comments** – Add via the "Comment" button. No post-it background; move freely. Double-click to edit.
- **Hand-drawn arrows** – Click "Connect", then click a note or comment, then click another to draw a dashed arrow between them.
- **Pages** – Multiple pages from the "Pages" dropdown. Add a page or switch between them.
- **Templates** – Five workflows: Sales Funnel, Brainstorm, Offer Map, Kanban, Mind Map. Each adds a new page with pre-filled notes and arrows.
- **Dark / light mode** – Toggle in the top-right (sun/moon).
- **Global font** – "Font" in the toolbar to switch between Google Fonts (DM Sans, Inter, Source Sans 3, Outfit, Plus Jakarta Sans, Playfair Display).
- **Persistence** – All data is saved in the browser (localStorage). Reload keeps your board.
- **Undo / Redo** – Toolbar buttons and Ctrl+Z / Ctrl+Shift+Z / Ctrl+Y. Delete/Backspace removes the selected note or comment.

